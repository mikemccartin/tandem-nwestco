## How to Use These Images

1. Visit the Unsplash URL
2. Download the appropriate size (Large recommended for hero, Medium for cards)
3. Save to `/current-site-assets/images/` folder
4. Update the HTML to reference the new image path

### Example for Option C

Replace gradient placeholders with actual images:

```css
/* Before */
.fuel-bg {
  background: linear-gradient(135deg, #25617C 0%, #3A7A98 100%);
}

/* After */
.fuel-bg {
  background: url('../../current-site-assets/images/fuel-systems-hero.jpg') center/cover no-repeat;
}
```
